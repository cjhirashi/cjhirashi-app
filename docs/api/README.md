# API Documentation

This directory contains documentation for API Routes and Server Actions.

## Structure

- **API Routes**: REST endpoints at `/api/admin/*` and `/api/agents/*`
- **Server Actions**: Form mutations with built-in CSRF protection
- **Response Format**: Standardized JSON with `apiSuccess()`, `apiError()` helpers
- **Validation**: Zod schemas for all inputs
