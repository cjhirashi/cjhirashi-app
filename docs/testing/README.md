# Testing Documentation

This directory contains documentation for Vitest unit tests and Playwright E2E tests.

## Structure

- **Unit Tests**: Vitest tests for utilities, helpers, and server actions
- **Integration Tests**: API route and database integration tests
- **E2E Tests**: Playwright end-to-end tests for critical user flows
- **Test Coverage**: Coverage reports and requirements
